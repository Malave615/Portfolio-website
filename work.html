Meet App

A Progressive Web App that allows users to discover, filter through, and track upcoming events in the city of their choice. It integrates with the google Calendar API and supports offline use, ensuring accessibility and usability anywhere. I was responsible both the frontend and the backend, using Node.js, React, Jest, Cucumber, Puppeteer, and Service workers to ensure seamless offline functionality and testing.

Pokedex App

A small web application that interacts with a public Pokémon API. Designed to teach API interaction by displaying detailed Pokémon data from an external API. The app allows users to search by name or ID and displays the Pokémon's name, ID, type, and an image. Built using jQuery, Bootstrap, and modern JavaScript features, with polyfills for wide browser compatibility.

Full-Stack Movie App

This app enables movie enthusiasts to explore a curated catalog of films, create a user profile, and manage their favorites. Users can also view info on the director, genre, and actors of each movie. The backend is built with Node.js and Express, and the frontend with React and Bootstrap. It demonstrates secure user authentication and RESTful API consumption.
