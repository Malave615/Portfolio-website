Meet App

This is a PWA (Progressive Web App) that allows users to find and filter through upcoming events in the city of their choice. I was responsible for the frontend and backend. The app is integrated with the Google Calendar API to display the events and their data. The app's written in React.js and Node.js and uses Jest, Cucumber, Puppeteer and Service workers to ensure seamless user interaction even when offline.
